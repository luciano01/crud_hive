- [x] Read
- [x] Update

## For developers
> To clone locally
```
$ git clone git@github.com:luciano01/crud_hive.git
$ cd crud_hive
```
> Open the project in an editor, if you use VS Code, follow this
```
$ cd crud_hive
$ code .
```
> To run, follow this on terminal
```
$ flutter clean
$ flutter pub get
$ flutter run --release
```
